# SBored
  It is a sound board  

# Dependencies
  Voicemeter Banana  
  Virtual Audio VB-Cable

# Installation
  Download repo  
  Set "VB-Cable output" as a hardware input in Voicemeter  
  Route VB-Cable output to your virtual microphone output in Voicemeter

# Use
  Add soundclips to "sounds" folder  
  Run SBored.pyw
